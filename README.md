# Personal Finance Tracker

An intermediate Flutter project for tracking personal finances and improving my Flutter skills.

## Features
- Track income and expenses
- Categorize transactions
- View statistics with charts
- Budget management
- SQLite database integration
- State management with Provider

## Getting Started

This project uses Flutter. To run this project:
```bash
flutter pub get
flutter run
```

## Tech Stack
- Flutter
- SQLite (sqflite)
- Provider (State Management)
- fl_chart (Charts)
